Devops web deployment

Description
 A project for deplying a webpage, with http -> redirection, self singed ssl certificate, backup and roll-back option

Getting Started

Dependencies
- nginx

 otherwise POSIX-complaient - don't worry we got you covered 

Installing
 use gitclone to have a working version of the project in your machine

Executing program
    run deploy.sh to deply the website, run backup.sh to create a backup. run health_check.sh to make sure everything is good, and if it isn't run roll-back.sh to go back to the previous version. Note that we only keep the last 5 versions, you greedy bastard
